  • Diseño Atomico

    • se basa en ir de lo mas basico e indivisible, atomos, que en este caso son las etiquetas

    a las moleculas, que son uniones de atomos, uniones de elementos basicos para formar una composicion

    y posteriormente estructuras hechas de moleculas y utilizar estas estructuras para formar parte de otras mas grandes

    la idea es crear estructuras, no paginas

Modular Scale

type Scale

para escalar la tipografia se usa un ratio

ratio: 1.618

base: 1em

h3: base * ratio

h2: h3 * ratio

h1: h2 * ratio

small: base / ratio

extra-small: small / ratio
